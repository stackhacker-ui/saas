## Quick Start

You can start a fresh new project with:

```bash [Terminal]
pnpm dlx nuxi@latest init my-saas -t gh:stackhacker-ui/saas
```

or create a new repository from GitHub:

1. Open <https://github.com/stackhacker-ui/saas>
2. Click on `Use this template` button
3. Enter repository name and click on `Create repository from template` button
4. Clone your new repository
5. Install dependencies with your favorite package manager
6. Start development server

That's it. You can now customize the SaaS pages and content in the [`content/`](https://content.nuxt.com/usage/content-directory) directory.
