This template is a SaaS starting point made with [Stackhacker UI](https://ui.stackhacker.io), shadcn-vue, Nuxt, and Tailwind CSS v4.

This template includes a customizable landing page, pricing page, documentation, blog, changelog, and authentication page UI for login and signup.

## Key Features

This template includes a range of features designed to accelerate SaaS app setup:

- **Powered by** [**Nuxt**](https://nuxt.com): Utilizes the latest Nuxt framework for optimal performance.
- **Built with** [**Stackhacker UI**](https://ui.stackhacker.io): Uses editable shadcn-vue primitives and local template blocks.
- [**MDC Syntax**](https://content.nuxt.com/usage/markdown) **via** [**Nuxt Content**](https://content.nuxt.com): Supports Markdown with component integration for dynamic content.
- **Content Navigation**: Generates docs navigation from the content structure.
- **Full-Text Search**: Includes built-in search functionality for content discovery.
- **Optimized Typography**: Features refined typography for enhanced readability.
- **Dark Mode**: Offers dark mode support for user preference.
- **Template Surfaces**: Includes marketing, pricing, content, changelog, and auth UI pages ready for customization.
