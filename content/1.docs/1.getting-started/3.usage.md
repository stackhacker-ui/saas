This is a practical example of what you can achieve with [Stackhacker UI](https://ui.stackhacker.io). You can tweak the template to match your product. The template uses [`@nuxt/content`](https://content.nuxt.com) for documentation, blog, changelog, and page content.

## Writing content

You can start writing `.md` or `.yml` files in the [`content/`](https://content.nuxt.com/usage/content-directory) directory to update the pages. Docs navigation is generated from the content tree, and the docs layout includes search data generated from the same content.
